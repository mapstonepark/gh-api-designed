A General Hospital API

This API will give users the following information about a character:

  • Actor or Actress
  • Current significant other or budding relationship
  • Current storyline

All objects returned will be in the following format so you can use dot notation to retrieve specific information.

The query parameter accepted is the name of any current contract character on General Hospital. An example of pulling Curtisâ€™s information would look like this:

https://generalhospitalapi.herokuapp.com/api/curtis ashford
